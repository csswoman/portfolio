# Portfolio - Karla Agraz

UX Engineer portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS** and **Storybook**.

## 🚀 Tech Stack

- **Next.js 16** & **React 19** - Modern web framework
- **TypeScript 6** - Type-safe development
- **Tailwind CSS 4** - Utility-first styling (CSS-first config)
- **Storybook 10** - Component documentation
- **next-seo** - SEO optimization
- **next-intl** - Internationalization
- **react-github-calendar** - GitHub activity display
- **ESLint 9** (flat config) - Code linting
- **PostCSS** & **@tailwindcss/postcss** - CSS processing

## ✨ Features

- 🎨 Responsive & accessible design
- 🔍 SEO optimized with Open Graph & Twitter Cards
- ⚡ Performance-focused (Lighthouse 95-100)
- 📚 Documented component library with Storybook
- 🌐 Internationalization support

## 📁 Project Structure

```
portfolio/
├── components/
│   ├── layout/      # Header, Footer
│   ├── sections/    # Hero, Work, About, Contact
│   └── ui/          # Reusable UI components
├── lib/
│   ├── constants.ts # Static data
│   └── seo.config.ts
├── pages/           # Next.js pages
├── public/          # Assets (images, fonts, icons)
├── styles/          # Global styles
└── types/           # TypeScript definitions
```

## 🛠️ Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# View components in Storybook
npm run storybook

# Build for production
npm run build
```
